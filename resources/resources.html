First Useful Website: W3_School

W3 school is a website that you can find resources on multiple coding language, from python to html to C++.

W3 school has a intuitive navigation system that you can check the navigation to find which topic you want to learn. In addition, it provides many examples for each topic that you can run and test the examples. In addition, you can modify the codes to see how everything works.

Second Useful Website: CodePen

CodePen is a website that provide real-time HTML,CSS and JS write that you can see the results instantly.

There are lots of useful codes on codePen that other users shared in it. It is very interesting to get some inspirations from other ideas and test it by exploring their code.
